## 🎯 Course Objective

The course has been designed to acquaint the students with the basic Object Oriented concepts and their uses using C++.  

The students will grasp how these principles contribute to creating **modular, reusable, and maintainable code**.  

The course aims to equip learners with the skills to:
- Design and implement complex software systems  
- Model real-world entities as objects  
- Solve programming problems efficiently  
- Develop robust applications  

---

## 📚 Course Content

### 🔹 Unit-I
**Principles of Object Oriented Programming**
- Concepts of Procedure Oriented Programming  
- Concepts of Object Oriented Programming  
- Object Oriented principles  

**Basics of C++**
- Keywords, Identifiers and Constants  
- Basic Data Types  
- User Defined Data Types  
- Derived Data Types  
- Variables and Reference Variables  
- Operators  

---

### 🔹 Unit-II
**Classes and Objects**
- Class and Defining Class Members  
- Objects  
- Memory Allocation of Objects  
- Objects as Function Arguments  
- Returning Objects  

**Functions in C++**
- Functions and their Benefits  
- Function Prototyping  
- Calling a Function  
- Inline Functions  
- Friend Function  
- Function Overloading  

---

### 🔹 Unit-III
**Constructors and Destructors**
- Constructors  
- Parameterized Constructors  
- Copy Constructors  
- Destructors  
- Static Data Members  
- Static Member Functions  

**Exception Handling**
- Exception Handling Mechanism  
- Throwing Mechanism  
- Catching Mechanism  
- Re-throwing an Exception  

---

### 🔹 Unit-IV
**Inheritance**
- Concept of Inheritance  
- Defining Derived Class  
- Types of Inheritance:
  - Single  
  - Multiple  
  - Multilevel  
  - Hierarchical  
  - Hybrid  
- Access Specifiers  
- Abstract Class  

---

## 🎓 Course Outcomes

- **CO1:** Understand the concepts of Procedure Oriented Programming and Object Oriented Programming, benefits of OOPs and its applications.  
- **CO2:** Understand the role of constants, variables, operators & the use of functions in C++.  
- **CO3:** Understand the use of constructors, destructors and exception handling.  
- **CO4:** Understand the concepts of inheritance in achieving code reusability.
